# React_Expense_App

An application for expenses built with React, JavaScript, and CSS.

-----

##### Download:

`$ git clone https://github.com/samira1992/React_Expense_App.git`

`$ cd path to the folder`

`$ npm install `


-----------
### Running the code
`npm start`
